# BakingApp

#### BakingApp is an Android app for learning desserts recipes step by step.

#### The app comes with a widget which gives you the option of showing the ingredients of a selected recipe on your home screen.

## Getting Started

#### These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites

#### BakingApp was developed using Android Studio IDE so you must install it in your computer before proceeding:

###### https://developer.android.com/studio/

## Next Steps

#### You can directly proceed to clone the project to your local machine, enter Android Studio, import and build your project and start using BakingApp on your Android device.

## Compatibility

#### Minimum Android SDK: BakingApp requires a minimum API level of 15.
#### Compile Android SDK: BakingApp requires you to compile against API 27 or later.
